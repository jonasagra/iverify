# iVerify 
Aplicação web para verificar o modelo do iPhone. 
Base coletada da antiga [wiki do iPhone](https://www.theiphonewiki.com/wiki/models).
Atualmente, contém mais de 2.000 modelos com nomes internos cadastrados, onde podemos adicionar ainda mais no banco de dados interno da aplicação.

# Características
- Imagens do modelo
- País em que o iPhone foi fabricado
- Tipo de modelo
