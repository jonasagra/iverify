# iVerify 
Aplicação web para verificar o modelo do iPhone. 
Base coletada da antiga [wiki do iPhone](https://www.theiphonewiki.com/wiki/models).
Atualmemte mais de 2.000 modelos de nomes internos cadastrados, podendo incluir ainda mais no banco de daods interno da aplicação.

# Características
- Imagens do modelo
- País em que o iPhone foi fabricado
- Tipo de modelo
